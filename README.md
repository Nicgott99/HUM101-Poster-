# HUM101 Academic Poster - Rise of Islamic Philosophy

An academic poster designed for the HUM101 (Introduction to Western Philosophy & Culture) course at BRAC University. The poster explores the **Rise of Islamic Philosophy and Its Spread to South Asia**.

## Overview

This project contains HTML-based academic posters created for a course presentation:

- **index.html** - A1 print-ready landscape poster formatted for physical printing
- **ENHANCED_POSTER.html** - Enhanced digital version with improved visuals and animations
- **PRINTABLE_A1_POSTER.html** - Optimized print layout for A1 paper

## Topics Covered

- Origins of Islamic Philosophy (8th-13th century)
- Key philosophers: Al-Kindi, Al-Farabi, Ibn Sina, Al-Ghazali, Ibn Rushd
- Transmission to South Asia through trade, Sufi mystics, and the Delhi Sultanate
- Integration with local philosophical traditions
- Lasting impact on South Asian intellectual life
